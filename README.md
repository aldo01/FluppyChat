# FluppyChat

This app is mobile messanger, like "whats app" or "viber". But with small modification. I added new essence "room".
Room is place where people can speaking in groups.

## Features

Application has few features:
- push notification
- "parse" api
- optimistic approach
- photo uploading
- material design
etc.
